Загрузка документов из Excel.

Это готовое решение для автоматического импорта данных из файлов Excel в систему электронного документооборота. Оно позволяет быстро загружать и обрабатывать документы, минимизируя ручной ввод и снижая вероятность ошибок.

Заходим на главную страницу и выбираем раздел Hippo ЭДО.

В разделе Hippo ЭДО переходим в Excel реестры.

На странице Excel реестры выбираем кнопку Массовое создание счет-фактур или Массовое создание накладных.

Загрузите ваш файл Excel и выберите нужные поля для соответствующих столбцов.

Выбираете нужный тип счета-фактуры из списка и нажимаете кнопку Синхронизация.

После успешной загрузки данных нажимается кнопка Массовая подпись.
